# xuborong.github.io

Personal website rebuilt with Astro and deployed via GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build and checks

```bash
npm run build
npm run check:links
```

## Content

- Blog posts: `src/content/blog/*`
- Pages: `src/content/pages/*`
- Navigation: `src/data/navigation.ts`
- Travel markers: `src/data/travel.ts`
